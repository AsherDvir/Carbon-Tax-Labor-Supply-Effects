# Carbon-Tax-Labor-Supply-Effects
Please download and unzip the data files. I use keka, which will automatically merge the zipped files as they originally were. 
After unzipping the data files, please change the directory in the stata do file so that the do file will appropriately read the data file. 
Tables 2,3, and 4 in the writing sample and the excel file, where the writing sample tables are were created, are from this stata do file. The first table was replicated from Table 7 in Boyce and Riddle (2007).
